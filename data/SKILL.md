## Curriculum Database Schema

The file `textbook-curriculum.csv` must follow this exact schema:

| Column | Type | Example |
|---|---|---|
| `grade` | integer | `2` |
| `subject` | string | `English` |
| `textbook` | string | `PCTB English Grade 2` |
| `chapter` | integer | `1` |
| `page_start` | integer | `10` |
| `page_end` | integer | `14` |
| `topic` | string | `My Family` |
| `slo` | string | `Students can identify and name immediate family members using simple sentences` |

---

## PCTB Textbook Context

- **PCTB** = Punjab Curriculum and Textbook Board — publisher of government school textbooks in Punjab province
- **Target scope:** English, Grades 2–5
- **SLO** = Student Learning Outcome — the measurable objective tied to a specific textbook page range
- These textbooks are used in all government primary schools in Punjab, and serve as the reference curriculum for lesson plan generation

---

## SLO Writing Standard

A good SLO must be:
- **Measurable** — starts with an observable action verb: identify, name, write, read, describe, match, complete, use
- **Grade-appropriate** — language complexity fits the grade level
- **Single-period achievable** — completable in one 40-minute class
- **Page-linked** — tied to a specific textbook page range (`page_start` to `page_end`)

Bad SLO: "Students understand family." (Not measurable)
Good SLO: "Students can identify and name immediate family members using simple sentences." (Measurable, observable)

---

## How Curriculum Data Is Used

At lesson plan generation time, the system:
1. Receives teacher input: grade pair + subject + page number
2. Queries this table: `SELECT slo FROM textbook_curriculum WHERE grade = ? AND page_start <= ? AND page_end >= ?`
3. Injects the matched SLO into the AI prompt template
4. This ensures plans reference exact textbook content, not generic topics

---

## Common Tasks

- **Add a chapter** → Append one row per SLO to `textbook-curriculum.csv` following the schema above
- **Validate an SLO** → Check it starts with a measurable verb and is linked to a page range
- **Build from textbook PDF** → Place PDF in this folder, read chapter by chapter, extract topic + SLO per page range

---

## Gotcha

Start with Grade 2, Chapter 1. Validate with a coach before building out Grades 3–5. One bad SLO template will propagate across all plans for that chapter — quality over speed here.
